Exploits

GET /exploits/count

Description: Fetches the total number of exploits available.

{
  "exploitCount": <number_of_exploits>
}

GET /exploits/summary

Description: Provides a summary of exploits, including the count of detected and safe exploits.

{
  "detectedExploits": <number_of_detected_exploits>,
  "safeExploits": <number_of_safe_exploits>
}

GET /exploits/

Description: Returns a list of the latest exploits for both Windows and Mac platforms.

[ { "_id": "662ae7576238b7f2082f253b", "title": "Wave", "version": "v1", "updatedDate": "6/28/2024 at 08:18 AM EST", ... } ]

GET /exploits/mac

Description: Returns a list of the latest Mac-specific exploits.

[ { "_id": "662d08a3cfca23ab88d4bc16", "title": "Macsploit", "version": "v1.1-e", "updatedDate": "6/27/2024 at 12:21 PM EST", ... } ]

GET /exploits/windows

Description: Returns a list of the latest Windows-specific exploits.

[ { "_id": "662ae7576238b7f2082f253b", "title": "Wave", "version": "v1", "updatedDate": "6/28/2024 at 08:18 AM EST", ... } ]

Offsets

GET /offsets/

Description: Returns the current Roblox offsets in plain text format.

Plain text of Roblox offsets

GET /offsets/plain

Description: Returns Roblox offsets with the constexpr std::uint32_t prefix removed.

Plain text of cleaned Roblox offsets

GET /offsets/:name

Description: Retrieves a specific offset value by name.

{
  "offsetName": "<offset_value>"
}

Versions

GET /versions/latest

Description: Retrieves the latest available version for either Windows or Mac.

{
  "latestVersion": <latest_version>
}

GET /versions/allversions

Description: Retrieves all available versions.

[ { "version": "v1.0", "date": "2024-01-01" }, { "version": "v1.1", "date": "2024-02-01" } ]

Example Usage

Get Total Exploit Count

curl https://yourwebsite.com/api/exploits/count

Get Latest Exploits for Windows

curl https://yourwebsite.com/api/exploits/windows

Get Specific Roblox Offset

curl https://yourwebsite.com/api/offsets/Name
